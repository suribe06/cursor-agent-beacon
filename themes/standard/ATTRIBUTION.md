# Standard theme attribution

The **standard** theme GIFs are **original pixel art** generated by this project via `scripts/build_character_sprites.py` and exported with `scripts/export_standard_gifs.py`.

- **License:** MIT (same as the repository)
- **Author:** cursor-agent-beacon contributors

No third-party character assets are bundled in the default theme.

If you build a **custom** theme using external art, see [`ASSETS.md`](ASSETS.md) for optional CC0 reference packs and keep copyrighted assets in `themes/custom/` only.
