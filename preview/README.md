# Display preview

Interactive simulator for the 240×240 agent display.

## Open the simulator

```bash
xdg-open preview/display-simulator.html
```

Requires `preview/beacon-sprites.js` (generated by `scripts/export_standard_gifs.py`) and `themes/standard/assets/*.gif` for the **Show exported GIF** toggle.

## What it shows

Full-body pixel robot with **4 animated frames per state**. Toggle **Show exported GIF** to preview the bundled files from `themes/standard/assets/`.

| State key | Agent state | Animation |
|---|---|---|
| `sleeping` | `idle` | Slow breath + Zzz |
| `waiting` | `waiting` | Awake blink + antenna |
| `thinking` | `thinking` | Yellow thought dots |
| `running_shell` | `running_shell` | Walk + speed lines |
| `running_mcp` | `running_mcp` | Cyan tool link |
| `success` | `success` | Soft green smile |
| `error` | `error` | Shake then hold |
| `stop` | `stop` | Wave + confetti |

## Regenerate assets

```bash
pip install Pillow
python3 scripts/export_standard_gifs.py
```

Updates GIFs, `pixel-faces.json`, `manifest.json`, and `preview/beacon-sprites.js`.

## Controls

- Click any state on the left panel
- **Auto cycle** — rotates through all states
- **Round display** — round bezel (GC9A01 style)
- **Show exported GIF** — switch between canvas preview and bundled GIF files
